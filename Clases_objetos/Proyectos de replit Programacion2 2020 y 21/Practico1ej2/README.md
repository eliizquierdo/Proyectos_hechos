**Ejercicio 2 (CAI – Reducción de la fatiga de los estudiantes):**

Un problema que se desarrolla en los entornos CAI es la fatiga de los estudiantes. Este problema puede eliminarse si se varían las contestaciones de la computadora para mantener la atención del estudiante. Modifique el programa del ejercicio 1, de manera que se muestren diversos

comentarios para cada respuesta, de la siguiente manera:

_Contestaciones respuesta correcta:_

**¡Muy bien!**

**¡Excelente!**

**¡Buen trabajo!**

**¡Sigue así!**

_Contestaciones respuesta incorrecta:_

**No. Por favor intenta de nuevo.**

**Incorrecto. Intenta una vez más.**

**¡No te rindas!**

**No. Sigue intentando.**

Use la generación de números aleatorios para elegir un número entre 1 y 4 que se utilice para seleccionar una de las cuatro contestaciones apropiadas a cada respuesta correcta o incorrecta.

Use una instrucción switch para emitir las contestaciones.
