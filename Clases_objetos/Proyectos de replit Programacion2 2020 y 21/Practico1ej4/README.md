**Ejercicio 4 (CAI – Niveles de dificultad):**

En los ejercicios anteriores se desarrolló un programa de instrucción asistida por computadora para enseñar a un estudiante de primara cómo multiplicar.

Modifique el programa para que permita al usuario introducir un nivel de dificultad.

• **Fácil**: El programa debe utilizar sólo números de 1 dígito en los problemas.

• **Intermedio** : El programa debe utilizar un número de 1 dígito y otro de 2.

• **Difícil**: El programa debe utilizar sólo números de 2 dígitos.
