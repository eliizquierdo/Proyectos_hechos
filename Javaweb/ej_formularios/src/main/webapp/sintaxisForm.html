Formularios HTML — Guía Rápida

En la columna “Sintaxis (genérica)”, lo que está en <ANGULARES> es lo que debés sustituir.

Elemento / Atributo Para qué sirve Sintaxis (genérica) Ejemplo (real)
<form> Contiene el formulario; define destino y método. <form action="<URL>" method="<get|post>"> ... </form> <form action="procesar.jsp" method="post">
type="text" Campo de texto de una línea. <input type="text" name="<nombreCampo>"> <input type="text" name="usuario">
type="number" Número con límites e incrementos. <input type="number" name="<nombreCampo>" min="<número>" max="<número>" step="<número>"> <input type="number" name="edad" min="0" max="120" step="1">
type="password" Contraseña (oculta caracteres). <input type="password" name="<nombreCampo>"> <input type="password" name="clave" required>
type="email" Correo con validación de formato. <input type="email" name="<nombreCampo>"> <input type="email" name="correo" placeholder="correo@dominio.com">
type="date" Selector de fecha. <input type="date" name="<nombreCampo>"> <input type="date" name="fecha_nac">
type="radio" Selección única por grupo. <input type="radio" name="<grupo>" value="<valorX>"> <input type="radio" name="genero" value="M"> Masculino
type="checkbox" Selección múltiple. <input type="checkbox" name="<nombreCampo>" value="<valorX>"> <input type="checkbox" name="interes" value="musica"> Música
<select> / <option> Lista desplegable. <select name="<nombreCampo>"><option value="<valorX>"><Texto></option></select> <select name="pais"><option value="UY">Uruguay</option></select>
<textarea> Texto multilínea. <textarea name="<nombreCampo>" rows="<n>" cols="<n>"></textarea> <textarea name="comentarios" rows="4" cols="40"></textarea>
name Clave que recibe el servidor. name="<nombreCampo>" name="usuario"
id Identificador único en la página. id="<nombreID>" id="usuario"
value Valor por defecto / enviado. value="<valorPorDefecto>" value="rojo"
placeholder Texto guía en el campo. placeholder="<textoGuia>" placeholder="Ingrese su nombre"
required Hace el campo obligatorio. required <input type="email" required>
min / max Límites (números/fechas). min="<valorMin>" max="<valorMax>" min="1" max="10"
step Incremento para números. step="<valor>" step="0.5"
checked Marca por defecto (radio/checkbox). checked <input type="checkbox" checked>
selected Opción preseleccionada en select. selected <option value="UY" selected>Uruguay</option>
multiple Permite múltiples selecciones. <select name="<nombreCampo>" multiple>...</select> <select name="colores" multiple><option>Rojo</option></select>
type="file" Subida de archivos. <input type="file" name="<nombreCampo>"> <input type="file" name="foto">
type="submit" Envía el formulario. <input type="submit" value="<textoBoton>"> <input type="submit" value="Enviar">
type="reset" Limpia el formulario. <input type="reset" value="<textoBoton>"> <input type="reset" value="Borrar">
<button> Botón genérico o de envío. <button type="<button|submit>"><Texto></button> <button type="button" onclick="alert('Hola')">Click</button>
